## v1.4.0 更新内容

### 🐛 修复 GUI 打不开（核心修复）

**根因**：`plugin.yml` 没有注册 `/k` 命令，玩家输入 `/k` 会收到 "Unknown command"。

- 在 `plugin.yml` 正式注册 `/k` 命令和 `aip.gui` 权限
- 新建 `GuiCommand` 实现 `CommandExecutor`，废弃 `PlayerCommandPreprocessEvent` 黑魔法
- 修复 `GuiManager.openPlayerList` 实体失效时 NPE 崩溃（加 `isValid()` 检查）
- 用 `InventoryHolder` 标识 GUI 替代 `title.startsWith("§6")` 匹配
- 修复 `/k` 权限失败时未取消事件导致 "Unknown command" 提示
- 修正误导文案从"按 K 键"改为"输入 /k"

### 🛠️ 代码休整

- 抽取 `fillEmpty(inv, from, to)` 公共方法消除三处重复代码
- 用 `ChatColor.stripColor()` 替换所有链式 `replace("§e","")` 调用
- 用 `Map<UUID, AIPlayer> selectedAi` 上下文替代从标题反解析 aiName
- `executeCommand` 加 try-catch，命令失败时给玩家反馈
- 清理 `GuiListener` 中未使用的 import

### ✨ 10 个新功能

1. **AI 统计面板** - GUI 中查看对话次数/行走距离/击杀数/在线时长
2. `/aip history <ai> [page]` - 对话历史分页查看（10 条/页）
3. `/aip personality set <ai> <brave|timid|grumpy|gentle>` - 个性设置，影响 LLM Prompt
4. `/aip team <create|join|leave|disband|list>` - 队伍系统，同队 AI 不互相攻击
5. `/aip task <assign|cancel|status>` - 长期任务指派（build/gather/patrol/escort/farm）
6. `/aip relation <set|show|list>` - 关系图谱（friend/enemy/neutral），敌对才攻击
7. `/aip revive <ai>` - 死亡复活，保留对话历史和个性
8. `/aip schedule <add|list|clear>` - 日程作息，昼夜节律自动行为
9. `/aip mood <ai>` - 情绪系统（0-100），影响 LLM Prompt
10. `/aip deathlog <ai>` - 死亡日志（时间/原因/凶手）
